# Carbon Footprint Tracker (React + Vite)

## Quick start

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run dev server:
   ```bash
   npm run dev
   ```

3. Open the URL printed by Vite (usually http://localhost:5173)

## Notes
- This is a demo app with simplified emission factors (not for production use).
- Data is persisted to `localStorage`.
